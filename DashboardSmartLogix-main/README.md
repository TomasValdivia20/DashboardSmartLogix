ReadMe Dashboard:

En este readme encontraran todo lo relacionado con el Dashboard de SmartLogix:

El front end esta realizado en Html, css, js ademas de React para la conversion.

En este dashboard encontraras con un panel de control en donde podras encontrar distintos tipos de informacion
como los pedidos que hay en cada bodega, los pedidos que han sido enviados, entregados y los que estan siendo procesados.


Si lo corren en HTMl ocupar show preview o live server
